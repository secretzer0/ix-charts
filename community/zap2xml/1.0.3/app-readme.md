# zap2xml

zap2xml fetches TV guide data from Zap2it and outputs XMLTV files for use with DVR software.

## Features

- Automatically fetches TV guide data on a configurable schedule
- Outputs standard XMLTV format compatible with most DVR software
- Supports US OTA and cable lineups via Zap2it

## Getting Started

1. Configure your Lineup ID and Postal Code
2. Set the output file path and update interval
3. Point your DVR software to the XMLTV file location

## Finding Your Lineup ID

Visit [Zap2it](https://tvlistings.zap2it.com/) and search for your location to find your lineup ID.
